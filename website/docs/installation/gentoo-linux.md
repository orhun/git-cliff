# Gentoo Linux

If you are using Gentoo Linux, **git-cliff** can be installed from [gentoo-packages](https://packages.gentoo.org/packages/dev-vcs/git-cliff), using:

```bash
emerge git-cliff
```

::note

The package is (currently) only available for `amd64`(~) architecture.

:::
